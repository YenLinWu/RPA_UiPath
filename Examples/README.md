## Worldwide_Tax_Summaries_Online   
至 [Worldwide Tax Summaries Online](https://taxsummaries.pwc.com/) 網站查詢多個國家的稅率資訊   
- 將每個查詢結果，以列印另存 PDF 的方式存檔；   
- 在每個查詢結果的頁面中，擷取特定的稅率，且建立一彙整表總結每個國家的特定稅率。
